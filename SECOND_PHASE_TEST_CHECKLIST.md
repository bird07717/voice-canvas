# 第二阶段 Scene Planner 测试清单

## 场景生成

- [ ] 说“画一个海边日落”，生成太阳、海面、沙滩、云等对象。
- [ ] 说“画一个公园”，生成草地、树、小路、太阳等对象。
- [ ] 说“画一张生日贺卡”，生成文字、星星、装饰元素。
- [ ] 说“画一个城市夜景”，生成月亮、高楼、道路。
- [ ] 说“画一个森林小屋”，生成房子、树、太阳或云。

## 布局

- [ ] 对象没有全部堆在中心。
- [ ] 背景对象在底层。
- [ ] 前景对象在上层。
- [ ] 多个同类对象有错开。
- [ ] 对象没有明显越界。

## 编辑

- [ ] 生成场景后说“选中太阳”，太阳高亮。
- [ ] 说“把它变大”，太阳变大。
- [ ] 说“把树移到左边”，树移动。
- [ ] 说“把海变成蓝色”，海变色。
- [ ] 鼠标拖动对象后，语音继续编辑正确对象。

## 撤销和保存

- [ ] 生成场景后撤销，整个场景回退。
- [ ] 保存后刷新，场景不丢。
- [ ] 导出 PNG 正常。

## Fallback

- [ ] “画一个红色圆”仍走快速命令。
- [ ] “撤销/保存/导出”仍走快速命令。
- [ ] LLM 输出非法 JSON 时不会修改画布。
- [ ] 未配置 LLM 时给出明确提示。
