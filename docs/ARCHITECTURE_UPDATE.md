# 架构调整说明

## 📋 更新内容

### ✅ 已完成的调整

#### 1. **主页新增设置功能**

**位置变更**：
- ❌ 原来：LLM设置和百度ASR设置在画布页面的侧边栏
- ✅ 现在：移到首页的"设置"按钮中

**功能**：
- 点击首页右上角"设置"按钮
- 打开设置Modal弹窗
- 包含两个标签页：
  - **LLM模型配置**：配置OpenAI兼容的API
  - **百度语音识别**：配置百度ASR的API Key

**优势**：
- 统一的设置入口
- 集中管理系统配置
- 画布页面更专注于绘画功能

#### 2. **画布页面简化**

**新的布局**：
```
┌──────────────────────────────────────────────────┐
│  [返回] 画布标题  [选择模型▼] [撤销] [保存] [导出] │
├────────┬─────────────────────────────┬───────────┤
│        │                             │           │
│ 语音   │       Canvas 画布区          │  对话     │
│ 控制   │                             │  历史     │
│        │                             │           │
│ [开始] │                             │  用户:... │
│ [停止] │                             │  AI:...   │
│        │                             │           │
└────────┴─────────────────────────────┴───────────┘
│ 状态栏: 监听中... 识别文本: "画一个圆"           │
└──────────────────────────────────────────────────┘
```

**侧边栏变更**：
- ✅ 保留：语音控制面板
- ✅ 新增：头部显示当前识别方式（百度ASR/浏览器识别）
- ✅ 新增：提示信息（未配置时的引导）
- ❌ 移除：LLM设置标签
- ❌ 移除：百度ASR设置标签

**顶部栏变更**：
- ✅ 新增：模型选择下拉框（快速切换LLM）
- ✅ 显示当前激活的模型名称

#### 3. **用户体验优化**

**首次使用流程**：
1. 登录系统
2. 点击"设置"按钮
3. 配置LLM模型（必需）
4. 配置百度ASR（可选）
5. 创建或打开画布
6. 在头部选择要使用的模型
7. 开始语音绘画

**画布使用流程**：
1. 进入画布页面
2. 头部选择LLM模型
3. 侧边栏开始语音识别
4. 说出绘画命令
5. 观察画布变化

## 🎯 新的使用流程

### 初次配置（在首页完成）

```bash
1. 登录系统
   ↓
2. 点击右上角"设置"按钮
   ↓
3. 在"LLM模型配置"标签中：
   - 添加OpenAI API配置
   - 测试连接
   - 保存
   ↓
4. 在"百度语音识别"标签中：
   - 输入API Key和Secret Key（或使用预置配置）
   - 测试连接
   - 保存
   ↓
5. 关闭设置弹窗
```

### 日常使用（在画布中操作）

```bash
1. 创建或打开画布
   ↓
2. 在头部选择要使用的LLM模型
   ↓
3. 侧边栏点击"开始语音识别"
   ↓
4. 对着麦克风说话："画一个红色的圆"
   ↓
5. 观察画布实时变化
   ↓
6. 查看右侧对话历史
```
