# 第二阶段 Scene Planner 演示脚本

## 演示前准备

- 后端、前端、数据库服务已启动。
- 当前画布为空或已保存。
- 已配置可用的 OpenAI-compatible LLM。
- 浏览器麦克风或百度 ASR 可正常识别中文。

## 主演示流程

1. 说：“画一个海边日落”
   - 预期：状态显示正在规划完整场景。
   - 预期：画布渐进生成太阳、海面、沙滩、云、树等对象。

2. 说：“选中太阳”
   - 预期：太阳被高亮选中。

3. 说：“把它变大一点”
   - 预期：太阳放大。

4. 说：“右边加一朵云”
   - 预期：画布右侧追加云，不重新生成整个场景。

5. 说：“把海变成深蓝色”
   - 预期：海面对象改色。

6. 说：“撤销”
   - 预期：如果刚刚生成了完整场景，整个场景作为一个历史步骤回退；普通追加或编辑按单步回退。

7. 说：“画一张生日贺卡”
   - 预期：生成贺卡背景、生日快乐文字、星星和装饰元素。

8. 点击“保存”，再点击“导出 PNG”
   - 预期：保存成功，对话历史显示场景记录，PNG 正常导出。

## 快捷入口演示

也可以在语音面板点击快捷场景：

- 海边日落
- 公园
- 生日贺卡
- 城市夜景
- 森林小屋

这些按钮复用同一条语音命令链路，适合录屏时避免 ASR 波动。
